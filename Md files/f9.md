ğŸ”‚ do...while Loop
Executes code at least once, even if the condition is false.

js
Copy
Edit

let num = 2;
let i = 1;

do {
    total = total + i;
    i++;
} while(i <= num);

console.log(total); // Sum from 1 to 2

js
Copy
Edit

ğŸ” continue Keyword
Skips the current iteration and continues with the next one.

js
Copy
Edit

    if(j === 5){
        continue; // Skips 5
    }
    console.log(j);
}
